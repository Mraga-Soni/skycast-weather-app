# 🌦️ SkyCast – Smart Weather Forecast App

A dynamic and responsive weather web application that provides real-time weather updates, hourly forecasts, and weekly predictions with an intelligent and visually adaptive UI.

---

## 🚀 Live Demo

🔗 https://skycast-weather-app-woad.vercel.app/

---

## ✨ Features

* 🌍 Detects user location using IP-based geolocation (with fallback support)
* 🔎 Search cities with basic auto-suggestions
* 🌡️ Temperature unit toggle (°C / °F)
* 📅 Hourly and 7-day weather forecast
* 🕒 Timezone-aware live clock
* 🎯 Dynamic UI:

  * Weather-based icons
  * Background changes based on conditions (day/night)
* 📊 Weather highlights:

  * UV Index
  * Humidity
  * Wind Speed
  * Visibility
  * Approximate air quality indicator


---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** Visual Crossing Weather API
* **Deployment:** Vercel

2. Open the project folder
